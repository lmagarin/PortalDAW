En este primera práctica veremos como instalar un servidor web Apache y PHP en un Ubuntu Server ubicado sobre máquina virtual local:

Volver al Home

INSTALAR APACHE:

Escribimos en consola:

     # apt-get install apache2

Con el siguiente comando obtenemos la IP en la que se ha alojado nuestro servidor web apache:

     # ifconfig

Comprobamos si funciona abriendo un navegador con la IP obtenida. Debería aparecer información oficial de Apache.

INSTALACIÓN PHP:

     # apt-get install php5 libapache2-mod-php5

Reiniciamos el servidor web:

     # service apache2 restart

El directorio www por defecto es:

     var/www

Creando un archivo info.php y con el código que se describirá abajo podemos cambiar la página de información de Apache por otra de la configuración de PHP. Para crear el archivo:

     # nano var/www/info.php

Escribir lo siguiente, guardar (Ctrl+O) y cerrar (Ctrl+X):

<?php
phpinfo();
?>

Si volvemos a escribir la IP de nuestro servidor web seguido de info.php observaremos dicha configuración.

     # XXX.XXX.XXX.XXX/info.php

Tendría que aparecer toda la información de configuración del PHP y sus módulos instalados.

A continuación buscamos el archivo php.ini y configuramos “display errors” en modo ON. De esta manera obtendremos los mensajes de errores de sintaxis en el navegador.
