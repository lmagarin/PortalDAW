Preguntas teóricas

  1. Declaración recomendada para la creación de arrays en JavaScript

    2. var array = [];

  2. pop() y push()

    1. Utilidad

      2. Son métodos del objeto array para insertar, push(), y para extraer, pop(), elementos del final de un array.

    2. Modifican el array original?

      3. Si, resultará el mismo array con el elemento añadido o extraido.

    3. Devuelven algún valor?

      4. Si, pop() devuelve el elemento eliminado, y push() devuelve la nueva longitud del array.

  3. shift() y unshift()

    1. Utilidad

      2. Son métodos del objeto array para insertar, unshift(), y para extraer, shift(), elementos del principio de un array.

    2. Modifican el array original?

      3. Si, resultará el mismo array con el elemento añadido o extraido.

    3. Devuelven algún valor?

      4. Si, shift() devuelve el elemento eliminado y unshift() devuelve la nueva longitud del array.

  4. De los cuatro métodos anteriores, indica los recomendados para la inserción/eliminación de un elemento y por qué?

    5. Depende de la finalidad de la acción, si quieres trabajar con los elementos finales del array debes usar push() y pop(), mientras que para los primeros elementos se usa shift() y unshift(). Eso si, push() y pop() son más rápidos que los otros porque no tienen que reasignar posiciones al resto de los elementos que permanecen en el array.

  5. join() y split(), ¿son métodos de String o de Array?

    6. join() es un método de la clase Array, y split() es un método de la clase String.

  6. splice()

    1. Utilidad

      2. Sirve tanto para eliminar elementos de un array como para añadir o sustituir elementos ya existentes en el array.

    2. Indica significado del primer argumento admitido: índice, cantidad o contenido

      3. Es el índice desde el que actuará el método splice()

    3. Indica significado del segundo argumento admitido: índice, cantidad o contenido

      4. Es la cantidad de posiciones que se verán afectadas por el método splice()

    4. Indica significado del resto de argumentos admitidos: índice, cantidad o contenido

      5. El resto de argumentos hacen alusión al contenido, es decir, nuevos elementos a ser añadidos al array.

    5. Modifican el array original?

      6. Si, devuelven el nuevo array con las modificaciones efectuadas.

  7. slice()

    1. Utilidad

      2. Sirve tanto para seleccionar una serie de elementos en un array e insertarlos en otro array nuevo.

    2. Indica significado del primer argumento admitido: índice, cantidad o contenido

      3. Es el índice desde el que comenzar la operación de extracción, y será el primer elemento extraido.

    3. Indica significado del segundo argumento admitido: índice, cantidad o contenido

      4. Es el índice donde finalizará la extracción de elementos, y el elemento de este índice se excluye de dicha extracción.

    4. Indica significado del resto de argumentos admitidos: índice, cantidad o contenido

      5. No hay más argumentos.

    5. Modifican el array original?

      6. No, el array original no se altera.

  8. Un Array es un Object:

    1. Indica resultado de typeof [ ]

      2. typeof (con paréntesis o sin ellos) devuelve información acerca del tipo en forma de cadena. Con lo que para Array devolverá "object".

    2. Método forEach() de un array. Indica utilidad y si funciona en todos los navegadores. Entrega un pequeño código que demuestre su uso

      3. forEach() Realiza la acción que se le especifique como argumento para cada elemento de un array. Su sintaxis es: array.forEach(función[, argumento]). El método forEach no modifica directamente el array original.

      Como puede verse en la imagen, el método foreach() es compatible con todos los navegadores actualmente excepto Opera Mini.
